motor_driver.main ( )

Sets up the pins required to use the MotorDriver class with a L6206 H-bridge and an Arduino shield.

Then spins the motor one way at 50% torque for 2 seconds, stops for 2 seconds, then spins the motor the other way at 50% torque for 2 seconds, then stops and disables the motor.This main code is run if this file is the main program but won't run if this file is imported as a module by some other program.
